# Django Блог

Простой блог на Django с возможностью ставить лайки.

## Функции
- Список постов
- Детальная страница поста
- Лайки (без регистрации)
- Админка Django

## Стек
- Python 3.14
- Django 6.0.6
- SQLite
- HTML/CSS
